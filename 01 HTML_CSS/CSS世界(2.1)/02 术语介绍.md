# 属于介绍
```CSS
.div{
	width:99px;
	color:white;
}
```
## 属性与值
width、color是属性
99px、white是值
## 关键字
white类似的
## 变量
称为变量的比较有限
## 长度单位
em、ps
s、ms
rad、deg
## 功能符
url()、calc()
## 属性值
冒号后面的值，1px solid #000 也是属性值
z-index:1 的1也是属性值
## 声明
width:80px; 
属性名+属性值 
## 声明块
{
	width:99px;
	color:white;
}
{} 属性块
## 规则和规则集
.div{
	width:99px;
	color:white;
}
选择器+声明块 
## 选择器
类选择器(.)、id选择器(#)、属性选择器(`[title]{}`)、伪类选择器(:first-child)、伪元素选择器(::after)
## 关系选择器
后代选择器、相邻后代选择器、兄弟选择器、相邻兄弟选择器
## @规则
@font-face、@page等
